Settings¶

Opens a separate settings window. LinaQA is highly configurable. In addition to general settings each analysis has its own settings. The window can be opened by clicking the button on the Main Tool Bar or by selecting ‘Settings’ from the Edit Menu. The settings are organised into groups depending on the type of setting and the analysis it applies to. Settings consist of ‘key’:’value’ pairs. Only the setting value can be edited. Setting values are strictly controlled according to the type of setting. The available setting groups are:

User Settings

  • General
  • Window
  • Pydicom

Analysis settings

  • 2D Phantoms
  • 3D Phantoms
  • Gamma
  • Picket Fence
  • Simple Sensitivity
  • Spatial Resolution
  • Starshot
  • Tomographic Contrast
  • Tomographic Uniformity
  • VMAT
  • Winston-Lutz
