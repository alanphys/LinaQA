PydicomÂ¶

These contain customisable options for Pydicom. Available settings are:

  • Force: If true Pydicom will try and open a file as a DICOM file. Use this if the file is not fully DICOM compliant.

  • Scale factor: Default amount to rescale the image or image series by. Use with Scale Image

Not all settings have been implemented.
