QuartDVTÂ¶

Automatically analyzes DICOM images of a QuartDVT phantom. The images can be from a CT or CBCT. A PDF report of results is generated and saved in the image directory. The default phantom can be specified in 3D Phantoms settings. While the default settings are usually sufficient certain settings can be changed in 3D Phantoms settings.

Please see the Quart Module in the Pylinac documentation for more information.

Not all settings have been implemented.
