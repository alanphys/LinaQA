File is not a valid image file!¶

LinaQA was unable to open this file as a DICOM file or image file. If this is a machine log file you may still do the Machine Log Analysis. Otherwise the file may be corrupted. Try opening it in another DICOM or Image viewer. If you believe that you should be able to open this image you can try setting ‘Force’ to True under Pydicom and open the image again.
