Delete Tag¶

Deletes an existing DICOM tag. The tag must be selected. The tag can be deleted by clicking the button on the DICOM Tool Bar or by selecting “Delete tag’ from the Edit Menu.See DICOM Tool Bar for editing DICOM files.

Editing a DICOM file may corrupt it.
