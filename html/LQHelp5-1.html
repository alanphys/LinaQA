Uninstalling previous versions¶

It is not necessary to uninstall previous versions. Please note that the installation utility will not backup any settings or data files in the directory. You will need to do this yourself. To uninstall the program please follow the instructions below:

Under Windows¶

To remove LinaQA

Delete the files in the installation directory. If you have any desktop shortcuts or menu links delete those as well

To remove Ana/Miniconda

In Windows open the ‘Control Panel’ from the main menu, double click ‘Programs and Features’, (previously known as ‘Add Programs’) and click on the ‘Anaconda’ or ‘Miniconda’ entry. Select ‘Uninstall’ or ‘Remove’. Any user files such as settings may need to be removed manually.

Under Linux¶

Delete the files in the installation directory. If you have any desktop shortcuts or menu links delete those as well. Also delete the directory ~/.config/YenzakahleMPI/LinaQA.conf and any files in it.
