Scale Image¶

Scales the image values by ‘Scale Factor’. The image can be inverted by clicking the button on the Radiology Tool Bar or by selecting ‘Scale’ from the Image Menu. This is a destructive operation, i.e. it changes the image data. The ‘Scale Factor’ can be set from the Radiology Tool Bar or in Pydicom.

Image data will be changed. Save the image to make the changes permanent.
