2D Phantom Analysis¶

Analyse single images acquired from 2D image quality phantoms. The analysis can be opened by clicking the button on the Radiotherapy Tool Bar or by selecting ‘Radiotherapy, 2D Phantoms’ from the Analyse Menu. The phantom type can be selected from the list box underneath the button. The default phantom can be specified in 2D Phantoms settings. If the image is inverted the invert flag is set and passed to the analysis. The 2D phantoms that are currently implemented are:

  • Doselab MC2 MV

  • Doselab MC2 kV

  • Doselab RLf

  • Elekta Las Vegas

  • IBA Primus A

  • IMT L-Rad

  • PTW Iso-Align

  • Las Vegas

  • Leeds

  • Leeds (Blue)

  • PTW EPID QC

  • SNC FSQA

  • SNC MV-QA

  • SNC MV-QA (12510)

  • SNC kV-QA

  • SI FC-2

  • SI QC-3

  • SI QC-kV]

While the default settings are usually sufficient certain settings can be changed as given in 2D Phantoms settings. Please see the Planar Imaging Module in the Pylinac documentation for more information.
