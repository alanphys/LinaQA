Raw/LUT
=======

Toggles display between raw pixel values and calibrated values if they exist. The image is then :ref:`autowindowed<autowindow>` The image display can be toggled by clicking the |scale| button on the :ref:`dxtoolbar` or by selecting 'Raw/LUT' from the :ref:`imagemenu`. This is a non-destructive operation, i.e. it does not change the image data.
