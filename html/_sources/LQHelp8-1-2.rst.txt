Open Ref
========

Opens a dialogue box to select a DICOM reference image to load. The main image is compared against this reference image in certain of the analyses. This function can be accessed using the 'Open reference image' button |open| on :ref:`maintoolbar` or by selecting 'Open Ref' from the :ref:`filemenu`.

|Note| If you do not see the file you want make sure you have selected the correct file type.
