DICOM Tool Bar
==============

The DICOM tool bar provides methods to edit DICOM files. The toolbar can be enabled by clicking the |dcm| button on the :ref:`maintoolbar` or by selecting "DICOM Tags' from the :ref:`editmenu`.

More functions can be found on the :ref:`dicomtab`. The DICOM file reading and writing is done via `Pydicom <https://pydicom.github.io/pydicom/stable/>`_. For trouble shooting please see the `Pydicom documentation <https://pydicom.github.io/pydicom/stable/>`_.
