Tomographic Contrast Analysis
=============================

Calculates the contrast ratios of different size spheres in a SPECT or PET image. The analysis can be opened by clicking the |tc| button on the :ref:`nmtoolbar` or by selecting 'Tomographic Contrast' from the :ref:`nucmedmenu` analysis menu. Currently works with the Jaszack phantom only.

Available settings can be set in :ref:`tomocontrastsettings` settings.

Please see `Tomographic Contrast <https://pylinac.readthedocs.io/en/latest/nuclear.html#tomographic-contrast>`_ in the `Pylinac documentation <https://pylinac.readthedocs.io/en/latest/>`_ for more information.
