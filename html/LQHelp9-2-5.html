Machine Log Analysis¶

Analyse Varian linear accelerator machine logs. The analysis can be opened by clicking the button on the Radiotherapy Tool Bar or by selecting ‘Radiotherapy, Machine Logs’ from the Analyse Menu. There are currently no settings associated with the Machine Log analysis.

Please see the Log Analyzer Module in the Pylinac documentation for more information.

Machine log files are neither image nor DICOM files and so will generate a warning when you open them. However, they can still be analysed.
