DICOM Tool Bar¶

The DICOM tool bar provides methods to edit DICOM files. The toolbar can be enabled by clicking the button on the Main Tool Bar or by selecting “DICOM Tags’ from the Edit Menu.

More functions can be found on the DICOM tags Tab. The DICOM file reading and writing is done via Pydicom. For trouble shooting please see the Pydicom documentation.
