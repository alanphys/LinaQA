Auto window¶

Rescales the window level and width on the Image Tab so that the image maximum is the brightest displayable value (white) and the image minimum is the darkest displayable value (black). The window level and width can be rescaled by clicking the button on the Radiology Tool Bar or by selecting ‘Auto window’ from the Image Menu. This is a non-destructive operation, i.e. it does not change the image data.
