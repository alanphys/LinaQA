Nuclear Medicine Tool BarÂ¶

The Nuclear Medicine tool bar provides methods to analyse DICOM images from static Gamma cameras, SPECT and PET. The analysis is done via the Nuclear Module and results are returned as a PDF report. No checking is done by the tests to see if the images are suitable. For trouble shooting please see the Pylinac documentation.
