The menu bar allows you to carry out operations on images and datasets. Some functions are duplicated on Main Tool Bar. Menu options can also be invoked using shortcut keys by pressing <alt> and the underlined character. Available menu items are:

  • File Menu
  • Edit Menu
  • Image Menu
  • Analyse Menu
  • Help Menu
  • Exit Menu
