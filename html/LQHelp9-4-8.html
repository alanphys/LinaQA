Tomographic Contrast Analysis¶

Calculates the contrast ratios of different size spheres in a SPECT or PET image. The analysis can be opened by clicking the button on the Nuclear Medicine Tool Bar or by selecting ‘Tomographic Contrast’ from the Nuclear Medicine analysis menu. Currently works with the Jaszack phantom only.

Available settings can be set in Tomographic Contrast settings.

Please see Tomographic Contrast in the Pylinac documentation for more information.
