Spatial Resolution Analysis¶

Calculates the spatial resolution of a Gamma Camera image.The analysis can be opened by clicking the button on the Nuclear Medicine Tool Bar or by selecting ‘Spatial Resolution’ from the Nuclear Medicine analysis menu. The spatial resolution test type can be selected from the list box below the button. The default test can be specified in Spatial Resolution. Currently available test types are:

  • Four Bar Spatial Resolution Analysis
  • Quadrant Spatial Resolution Analysis

While the default settings are usually sufficient certain settings can be changed as given in Spatial Resolution.
