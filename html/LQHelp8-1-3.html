Save¶

Saves the current DICOM file. This function can be accessed using the ‘Save image’ button on Main Tool Bar or by selecting ‘Save’ from the File Menu.

This will overwrite the existing file. If changes have been made and the existing file should be kept use Save As. Files are stored internally uncompressed and will be saved uncompressed.
