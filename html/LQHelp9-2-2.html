Picket Fence Analysis¶

Analyse a picket fence Multi-Leaf Collimator (MLC) test pattern. The analysis can be opened by clicking the button on the Radiotherapy Tool Bar or by selecting ‘Radiotherapy, Picket Fence’ from the Analyse Menu. The MLC type can be selected from the list box underneath the button. The default MLC can be specified in Picket Fence settings. MLC’s that are currently implemented are:

  • Varian Millenium 120 leaf

  • Varian HD Millenium 120 leaf

  • B Mod 80 leaf

  • Elekta Agility 160 leaf

  • Elekta iMLC 80 leaf

  • Varian Halcyon Proximal and Distal 28 leaf.

While the default settings are usually sufficient certain settings can be changed as given in Picket Fence settings. Please see the Picket Fence Module in the Pylinac documentation for more information.
