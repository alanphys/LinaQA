VMAT Analysis¶

Analyse images acquired during Varian’s Dose Rate - Gantry Speed and MLC Leaf Speed tests. The analysis can be opened by clicking the button on the Radiotherapy Tool Bar or by selecting ‘Radiotherapy, Picket Fence’ from the Analyse Menu. The speed test type can be selected from the list box underneath the button. The default test can be specified in VMAT settings. The tests that are currently implemented are:

  • Dose Rate - Gantry Speed (DRGS)

  • Dose Rate - MLC Leaf Speed (DRMLC)

While the default settings are usually sufficient certain settings can be changed as given in VMAT settings. Please see the VMAT Module in the Pylinac documentation for more information.
