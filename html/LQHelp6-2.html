Running LinaQA under LinuxÂ¶

Open a terminal window. Change to the directory where you installed LinaQA. At the prompt enter:

python LinaQA.pyw

or

python LinaQA.pyw \path\to\dicom\image.dcm

or

python LinaQA.pyw \path\to\dicom\imagedir

Alternatively you can create a link to the application and place it on your desktop. Refer to your system documentation for details.
