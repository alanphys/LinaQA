Radiotherapy Tool Bar
=====================

The Radiotherapy tool bar provides methods to analyse EPID images and kV images. The analysis is done via `Pylinac <https://pylinac.readthedocs.io/en/latest/>`_ and results are returned as a PDF report. No checking is done by the tests to see if the images or files are suitable. For trouble shooting please see the `Pylinac documentation <https://pylinac.readthedocs.io/en/latest/>`_.
